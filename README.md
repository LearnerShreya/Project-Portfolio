## üöÄ Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation
```bash
# Clone the repository
git clone <your-repo-url>
cd portfolio-nextjs

# Install dependencies
npm install

# Run development server
npm run dev
```

### Build for Production
```bash
# Build the application
npm run build

# Start production server
npm start
```
